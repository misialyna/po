# Programowanie Obiektowe — WAT

Materiały do kursu **Programowanie Obiektowe** na Wydziale Cybernetyki WAT (por. mgr inż. Norbert Waszkowiak).

## Zawartość

### Prezentacja wykładowa (`index.html`)

Slajdy reveal.js pokrywające: podstawy C/C++, tablice, wskaźniki, klasy i obiekty, dziedziczenie, szablony i STL.

Uruchomienie:
```bash
npm install
npm start
# lub po prostu otwórz index.html w przeglądarce
```

### Ćwiczenia laboratoryjne (`exercises/`)

Implementacje w C++ z zajęć laboratoryjnych:

| Folder | Klasy | Zagadnienie |
|---|---|---|
| `article-author/` | `Article`, `Author` | Relacje między klasami, kompozycja |
| `list/` | `List` | Implementacja listy |
| `geometry/` | `Point`, `Triangle` | Geometria, konstruktory, metody |
| `patterns/` | `MyArray`, `MySingleton` | Szablony tablic, wzorzec Singleton |
| `products/` | `MineralWater`, `Notebook`, `Product` | Hierarchia produktów, dziedziczenie |
| `workers/` | `Employee`, `Manager`, `Programmer`, `TeamLeader` | Hierarchia pracowników, polimorfizm |
